🏃 Web stranica trkačkog kluba

Moderna, responzivna i profesionalna web stranica za trkački klub izrađena pomoću HTML-a, CSS-a i JavaScript-a. Projekat je fokusiran na moderan klupski izgled, preglednu strukturu, interaktivne elemente i dobar prikaz na desktop i mobilnim uređajima.

---

🌟 Opis projekta

Ova web stranica predstavlja trkački klub kroz moderan vizuelni identitet, informacije o klubu, ruti, utrci, galeriji i online prijavi. Dizajn je prilagođen korisnicima i uključuje light/dark temu, countdown do utrke, FAQ sekciju i profesionalne UI detalje.

---

🛠️ Tehnologije

- 🧱 HTML5
- 🎨 CSS3
- ⚙️ JavaScript
- 📱 Responsive Design
- 🌙 Light/Dark Theme
- ✨ UI animacije i ikone

---

🚀 Funkcionalnosti

- 🏠 Moderna početna stranica
- ⏳ Countdown do naredne utrke
- 🗺️ Prikaz rute i informacija o vožnji
- 🖼️ Foto galerija
- 📝 Forma za prijavu
- ❓ FAQ accordion sekcija
- 🌗 Light/Dark tema
- 📱 Prilagođeno za mobitel, tablet i desktop
- 🎯 Glatke animacije i profesionalan UI izgled

🎨 Dizajn

Stranica koristi moderan klupski stil sa staklenim karticama, svijetloplavim detaljima, responzivnim grid rasporedom, hover efektima i čistim sekcijama. Poseban fokus je stavljen na preglednost countdown dijela, kartica događaja i forme za prijavu.

---

📱 Responzivnost

Web stranica je optimizovana za:

- 💻 Desktop uređaje
- 📱 Mobilne telefone
- 📟 Tablete

Layout se automatski prilagođava veličini ekrana kako bi sadržaj ostao pregledan i funkcionalan.

▶️ Pokretanje projekta

Projekat ne zahtijeva dodatnu instalaciju.

1. Preuzmi ili kloniraj repozitorij.
2. Otvori fajl "index.html" u browseru.
